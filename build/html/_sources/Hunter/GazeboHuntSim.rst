Gazebo Simulation
=============================


1. Enter hunter_ws folder

.. code-block:: text

	cd hunter_ws

2. Declare environment variables

.. code-block:: text

	source devel/setup.bash
	
3. Simulate

.. code-block:: text

	roslaunch hunter2_gazebo hunter2_gazebo.launch
