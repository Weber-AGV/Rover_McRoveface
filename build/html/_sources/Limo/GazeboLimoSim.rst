Gazebo Simulation
=============================

1. Enter AgileX_ws folder

.. code-block:: text

	cd AgielX_ws

2. Declare environment variables

.. code-block:: text

	source devel/setup.bash
	
3. Simulate

.. code-block:: text

	roslaunch limo_gazebo_sim limo_ackerman.launch
