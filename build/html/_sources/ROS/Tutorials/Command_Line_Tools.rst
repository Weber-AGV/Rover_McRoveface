DEBUG commands
================

The following are commands you can use to debug your ros network.


.. list-table:: commands
   :widths: 50 50
   :header-rows: 1

   * - Command
     - Description
   * - rosrun 
     - to debug nodes
   * - rosnode info
     - gives info on running node
   * - rosrun rqt_graph rqt_graph
     - shows the node graph of ROS
   * - rosnode kill
     - shut down running node
   * - rosnode ping
     - check if a node is in the graph
   * - rosnode list
     - shows active running nodes
   * - rostopic echo
     - print messages to screen
   * - rostopic info
     - print information about active topic
   * - rostopic list
     - list active topics   
   * - rostopic pub
     - publish data to topic   
   * - rosservice list
     - lists active services
   * - rosservice info
     - print information about service 
   * - rosservice call
     - call the service with the provided args
   * - rqt_graph
     - to visualize nodes and topics
