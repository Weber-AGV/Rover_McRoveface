==================
**Prerequisites**
==================

.. toctree::
   :hidden:
   :glob:

   Prerequisites/Ubuntu-Environment
   Prerequisites/Linux-Terminal
   Prerequisites/Python



Install the proper environment to complete software tutorials/ simulate AgileX products. We require **ROS Noetic** on **Ubuntu Linux**. If you don't already have Ubuntu Linux OS, you will need to install a **Virtual Machine**. This guide includes information on how to do these steps, along with information on basic **Python3** and **Linux terminal skills**.
