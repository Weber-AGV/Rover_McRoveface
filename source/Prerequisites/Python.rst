Python
==============

ROS can use both Python and C++, but for this course we will use Python. 

*Install*
----------

Instal Python3 with the following command in the Terminal

.. code-block:: text

	sudo apt install python3

You should be able to see your newlly installed python by executing the following command.

.. code-block:: text

	python3 --version
